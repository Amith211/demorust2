# demorust2

## My Second Rust project

### Takes in inputs from user changing method that is run

### Features include:
* Enums
* Generics
* Traits
* Vectors
* Iterators
* PhantomData
* Aliases
* Crate dependencies

## Run Syntax
```
demorust2 {Mode} {Value}
```
## Example
```sh
demorust2 Direct Hello
```
## Help
```sh
demorust2 Help
```
Will display help information and the available options

## Modes
* Direct
* Saver
